# KFC Theme

A modern WordPress block theme for Kingston Fencing Club.

## Description

KFC Theme is a custom block theme designed specifically for Kingston Fencing Club. It features:

- Modern, clean design with the KFC color scheme
- Full Site Editing (FSE) support
- WooCommerce integration for course bookings
- Custom patterns for hero sections, features, and team members
- Responsive design for mobile and desktop
- Social media integration
- Optimized for performance

## Features

### Patterns
- **Hero with Video** - Full-width hero with video background
- **Hero with Image** - Hero section with animated zoom effect
- **Features** - 2x2 grid showcasing club programs
- **Team Member** - Profile cards for coaches and staff

### WooCommerce Support
- Custom product pages
- Styled checkout and cart pages
- Course date display for products

### Custom Functionality
- Social media icons in footer
- Checkout waiver notice
- Course dates shortcode
- Forminator form styling
- MailPoet subscription form styling

## Installation

1. Download the `kfc-theme.zip` file
2. Go to WordPress Admin > Appearance > Themes
3. Click "Add New" > "Upload Theme"
4. Choose the ZIP file and click "Install Now"
5. Click "Activate" to use the theme

## Requirements

- WordPress 6.0 or higher
- PHP 7.4 or higher
- WooCommerce plugin (for course bookings)

## Configuration

### Colors
The theme uses the following color palette:
- Primary Blue: #0099ff
- Navy Blue: #0a1e3c
- Dark: #212529
- Light Gray: #f8f9fa
- White: #ffffff

### Fonts
- Primary Font: Kanit (Google Fonts)
- System Font: System UI fallback

### Social Media
Edit social media links in the footer via:
Appearance > Editor > Patterns > Footer
